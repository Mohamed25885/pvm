# Commands Directory

This directory contains reusable command templates for common operations.

## What are Command Templates?

Command templates are predefined command structures that agents can use and adapt for common operations like:
- Running tests
- Building the project
- Analyzing code
- Formatting code

## Usage

When executing common operations:
1. Check this directory for existing templates
2. Use the template as a starting point
3. Adapt arguments as needed

See individual files in this directory for specific command templates.
