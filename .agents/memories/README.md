# Memories Directory

This directory contains past interactions, learnings, and project context that agents should remember.

## What are Memories?

Memories store important information that agents should know when working on the project:
- Key decisions and their rationale
- Lessons learned from previous tasks
- Project-specific conventions
- Important context about the codebase

## Usage

When starting a new task:
1. Check this directory for relevant memories
2. Apply learned lessons to current work
3. Add new memories when significant learnings occur

## Adding Memories

When adding a new memory:
- Use a descriptive filename
- Include the date
- Document the context and what was learned
- Explain how it affects future work
