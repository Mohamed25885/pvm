# Skills Directory

This directory contains specialized skills for specific tasks.

## What are Skills?

Skills are reusable definitions that help agents perform specific tasks efficiently. They can include:
- Workflows for common operations
- Best practices for specific code types
- Validation rules and checklists

## Usage

When performing a task, check if a relevant skill exists:
1. Look for skill files in this directory
2. Follow the defined workflow
3. Adapt as needed for the specific task

## Creating New Skills

When creating a skill, document:
- What the skill does
- When to use it
- Steps to follow
- Common pitfalls to avoid
