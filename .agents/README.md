# .agents Directory

This directory contains resources for agentic coding agents working on the PVM project.

## Structure

| Directory | Purpose |
|-----------|---------|
| [plans/](plans/) | Plan documents for feature development and refactoring |
| [skills/](skills/) | Specialized skills for specific tasks |
| [commands/](commands/) | Reusable command templates |
| [templates/](templates/) | Code templates |
| [memories/](memories/) | Past interactions, learnings, project context |
| [docs/](docs/) | Documentation, guidelines, references |

## Usage

Before starting any task, agents MUST:

1. **Read this README** - Understand the .agents structure
2. **Check [plans/](plans/)** - Review existing plans for context
3. **Check [docs/](docs/)** - Read relevant guidelines and documentation
4. **Use [skills/](skills/)** - Apply specialized skills when available
5. **Use [templates/](templates/)** - Follow code templates for consistency
6. **Update [memories/](memories/)** - Record important learnings
