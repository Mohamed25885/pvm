# Templates Directory

This directory contains code templates for common patterns used in the PVM project.

## What are Templates?

Templates are predefined code structures that help maintain consistency across the codebase. They include:
- Class templates
- Command templates for CommandRunner
- Test templates
- Interface templates

## Usage

When creating new code:
1. Check this directory for relevant templates
2. Use the template as a starting point
3. Adapt to fit the specific requirements

## Available Templates

See individual files in this directory for specific code templates.
